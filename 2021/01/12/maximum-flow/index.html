maximum flow

2021-01-12 topics 1.4k words

introduction

a network is a graph \(G = (V, E)\) together with its capacity function \(c \colon V \times V \to [0, \infty]\).

a flow network \((G, c, s, t)\) is a network with two distinguished nodes: source \(s\) and sink \(t\).

below is an example of a flow network, with \(s = 1\) and \(t = 4\):

a flow is a function \(f \colon V \times V \to \mathbb R\) such that

  • \(f(u, v) = -f(v, u)\) for all nodes \(u\) and \(v\),
  • \(f(u, v) \leq c(u, v)\) for all nodes \(u\) and \(v\),
  • \(\sum_{(v, u) \in E} f(v, u) = \sum_{(u, v) \in E} f(u, v)\) for all nodes \(u\) except \(s\) and \(t\),
  • \(\sum_{u \in V} f(u, s) = 0\), and
  • \(\sum_{u \in V} f(t, u) = 0\).

in words,

  • the flow through each edge corresponds to a negative flow of the same magnitude in the antiparallel edge (the significance of which will be explained later),
  • the flow through each edge should not exceed its capacity,
  • the flow into a node that is not \(s\) or \(t\) should equal the flow out of the node,
  • there should be no flow into \(s\), and
  • there should be no flow out of \(t\)

the value \(\left | f \right |\) of a flow \(f\) is defined by \[ \left | f \right | = \sum_{u \in V} f(s, u) = \sum_{u \in V} f(u, t). \]

in words, this is the amount of flow out of \(s\) which is also the amount of flow into \(t\).

suppose we want to find a flow of maximum value in a given flow network. we can use the ford-fulkerson method to solve this problem.

ford-fulkerson method

the residual capacity \(r(u, v)\) of an edge \((u, v)\) is defined by \(r(u, v) = c(u, v) - f(u, v)\).

we can then construct the residual network \((G, r, s, t)\) of the flow network \((G, c, s, t)\).

an augmenting path is a simple path \(p = ((s, v_1), (v_1, v_2), \dots, (v_{n - 2}, t))\) from \(s\) to \(t\) in the residual network such that \(r(u, v) > 0\) for all \((u, v) \in p\).

the ford-fulkerson method works as follows:

  1. set \(f(u, v) = 0\) for all \(u, v \in V\).
  2. construct the residual network.
  3. find an augmenting path \(p\). if an augmenting path cannot be found, the flow is maximal and the algorithm ends.
  4. let \(inc = \min_{(u, v) \in p}(r(u, v))\). \(f(u, v) += inc\) and \(f(v, u) -= inc\) for all \((u, v) \in p\).
  5. go to step 2.

edmonds-karp algorithm

the edmonds-karp algorithm is an implementation of the ford-fulkerson method using bfs to find the shortest augmenting path at each step. when an augmenting path is found, residual capacities are updated by backtracking along the found path instead of reconstructing the entire residual network. the time complexity is \(\mathcal O(VE^2)\).

dinic's algorithm

dinic's algorithm is an implementation of the ford-fulkerson method using bfs to generate a level graph of the residual network and using dfs to find augmenting paths to reach a blocking flow at each step.

a blocking flow is a flow \(f\) in a network \(((V, E), c, s, t)\) such that there are no \(s-t\) paths in graph \(G' = (V, E')\) where \(E' = \left \{(u, v) ~ \colon ~ f(u, v) < c(u, v) \right \}\). in words, a blocking flow a flow such that no more augmenting paths can be found.

dinic's algorithm works as follows:

  1. assign to each node \(i\) a label \(label[i]\) equal to its depth in the bfs tree rooted at \(s\) considering only edges \((u, v)\) such that \(f(u, v) < c(u, v)\); if a node \(j\) is unreachable from \(s\), then \(label[j] = \infty\).
  2. generate level graph \(G_L(V, E_L)\) where \(E_L = \left \{ (u, v) ~ \colon ~ label[v] = label[u] + 1 \right \}\).
  3. if \(label[t] = \infty\), the flow is maximal and the algorithm ends
  4. run dfs on the level graph to find augmenting paths until a blocking flow is reached
  5. update the value of the flow and the residual capacities.
  6. go to step 1.

the time complexity is \(\mathcal O(V^2E)\).

implementations

edmonds-karp algorithm \(\mathcal O(VE^2)\) 
const int INF = 1e9 + 7;

int edmonds_karp_bfs(const vector<vector<int>> &graph, const vector<vector<int>> &capacity, int s, int t, vector<int> &backtrack) {
    fill(backtrack.begin(), backtrack.end(), -1);
    backtrack[s] = -2;

    queue<pair<int, int>> q;
    q.push({s, INF});

    while (!q.empty()) {
        int u = q.front().first, flow = q.front().second;
        q.pop();

        for (int v : graph[u]) {
            if (backtrack[v] == -1 && capacity[u][v] > 0) {
                // if unvisited and residual capacity is nonzero
                backtrack[v] = u;
                if (v == t) {
                    // augmenting path found
                    return min(flow, capacity[u][v]);
                } else {
                    q.push({v, min(flow, capacity[u][v])});
                }
            }
        }    
    }

    // augmenting path not found
    return 0;
}

// note that the contents of capacity may be modified
int edmonds_karp(const vector<vector<int>> &graph, vector<vector<int>> &capacity, int s, int t) {
    int flow = 0, inc;
    vector<int> backtrack(graph.size());

    while ((inc = edmonds_karp_bfs(graph, capacity, s, t, backtrack)) != 0) {
        // an augmenting path with value inc is found
        flow += inc;

        // backtrack along the augmenting path to update residual capacities
        for (int current = t, previous = backtrack[current]; current != s; current = previous, previous = backtrack[current]) {
            // update residual capacity
            capacity[previous][current] -= inc;
            capacity[current][previous] += inc;
        }
    }

    return flow;
}

dinic's algorithm \(\mathcal O(V^2E)\) 
const int INF = 1e9 + 7;

bool dinic_bfs(const vector<vector<int>> &graph, const vector<vector<int>> &capacity, int s, int t, vector<int> &label) {
    fill(label.begin(), label.end(), -1);
    label[s] = 0;

    queue<int> q;
    q.push(s);

    while (!q.empty()) {
        int u = q.front();
        q.pop();
        for (int v : graph[u]) {
            if (label[v] == -1 && capacity[u][v] > 0) {
                // if unvisited and residual capacity is nonzero
                label[v] = label[u] + 1;
                q.push(v);
            }
        }
    }

    return label[t] != -1;
}

// note that the contents of capacity may be modified
int dinic_dfs(const vector<vector<int>> &graph, vector<vector<int>> &capacity, int t, const vector<int> &label, int u, int flow) {
    if (u == t) {
        // augmenting path found
        return flow;
    }

    // available amount of flow that can be used in augmenting paths
    int available = flow;
    for (int v : graph[u]) {
        if (available == 0) {
            // no need to check for more augmenting paths
            break;
        }

        if (label[v] == label[u] + 1 && capacity[u][v] > 0) {
            int use = dinic_dfs(graph, capacity, t, label, v, min(capacity[u][v], available));
            available -= use;

            // update residual capacities
            capacity[u][v] -= use;
            capacity[v][u] += use;
        }
    }

    // return the total amount of flow used in all augmenting paths
    return flow - available;
}

int dinic(const vector<vector<int>> &graph, vector<vector<int>> &capacity, int s, int t) {
    int flow = 0;
    vector<int> label(graph.size());

    while (dinic_bfs(graph, capacity, s, t, label)) {
        flow += dinic_dfs(graph, capacity, t, label, s, INF);
    }

    return flow;
}
