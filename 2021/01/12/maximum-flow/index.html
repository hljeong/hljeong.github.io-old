introduction

a network is a graph \(G = (V, E)\) together with its capacity function \(c \colon V \times V \to [0, \infty]\).

a flow network \((G, c, s, t)\) is a network with two distinguished nodes: source \(s\) and sink \(t\).

below is an example of a flow network, with \(s = 1\) and \(t = 4\):

a flow is a function \(f \colon V \times V \to \mathbb R\) such that

  • \(f(u, v) = -f(v, u)\) for all nodes \(u\) and \(v\),
  • \(f(u, v) \leq c(u, v)\) for all nodes \(u\) and \(v\),
  • \(\sum_{(v, u) \in E} f(v, u) = \sum_{(u, v) \in E} f(u, v)\) for all nodes \(u\) except \(s\) and \(t\),
  • \(\sum_{u \in V} f(u, s) = 0\), and
  • \(\sum_{u \in V} f(t, u) = 0\).

below is an example of a flow in the above flow network:

the value \(\left | f \right |\) of a flow \(f\) is defined by \[ \left | f \right | = \sum_{u \in V} f(s, u) = \sum_{u \in V} f(u, t). \]

suppose we want to find a flow of maximum value in a given flow network. we can use the ford-fulkerson method to solve this problem.

ford-fulkerson method

the residual capacity \(r(u, v)\) of an edge \((u, v)\) is defined by \(r(u, v) = c(u, v) - f(u, v)\).

we can then construct the residual network \((G, r, s, t)\) of the flow network \((G, c, s, t)\).

an augmenting path is a simple path \(p = ((s, v_1), (v_1, v_2), \dots, (v_{n - 2}, t))\) from \(s\) to \(t\) in the residual network such that \(r(u, v) > 0\) for all \((u, v) \in p\).

the ford-fulkerson method works as follows:

  1. set \(f(u, v) = 0\) for all \(u, v \in V\).
  2. construct the residual network.
  3. find an augmenting path \(p\). if an augmenting path cannot be found, the flow is maximal and the algorithm ends.
  4. let \(inc = \min_{(u, v) \in p}(r(u, v))\). \(f(u, v) += inc\) and \(f(v, u) -= inc\) for all \((u, v) \in p\).
  5. go to step 2.
